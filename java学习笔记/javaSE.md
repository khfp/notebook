## length() 方法，length 属性和 size() 方法的区别:


* length() 方法是针对字符串来说的，要求一个字符串的长度就要用到它的length()方法；
* length 属性是针对 Java 中的数组来说的，要求数组的长度可以用其 length 属性；
* Java 中的 size() 方法是针对泛型集合说的, 如果想看这个泛型有多少个元素, 就调用此方法来查看!
